# Create New .NET Project

Help me create a new .NET project. Follow these steps:

1. **Ask what I need**: What type of project (web API, console, library, MAUI, test, etc.), framework version, and any specific features
2. **Search templates**: Use `template_search` to find matching templates
3. **Show options**: Present the top matches with brief descriptions
4. **Inspect chosen template**: Use `template_inspect` to show available parameters
5. **Preview**: Use `template_dry_run` with my chosen parameters to show what files will be created
6. **Create**: Use `template_instantiate` to scaffold the project
7. **Verify**: Build the project with `dotnet build` to confirm it works
